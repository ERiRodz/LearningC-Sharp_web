: Console Input and Output

    • Placeholder

      • The methods Console.Write() and Console.WriteLine() are defined as follow: public static void Write("string", object arg0, object arg1, ...)
        This brings to its first parameter (the string) the placehorlder {0}. Placeholdes indicates what object goes in its place in the string by the number between the { }; the number refers to the object's position. Placeholders can be formatted as: {index[,alignment][:format]}
        -index indicates the object
        -alignment: (right) +#; (left) -#; the # indicates the spaces
        -format (see table of specifiers)
