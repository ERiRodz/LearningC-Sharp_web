: Operators and Expressions

    • Operators

      • We use operators to perform differents actions on the data. Operators allow processing of primitive data and objects. They take as an input one or more operands and return some value as a result.


      • Category Operators Description
        arithmetic -, +, *, /, %, ++, -- Math operations
        logical &&, ||, !, ^ Boolean operations
        binary &, |, ~, ^, <<, >> Binary Represenatation operations
        comparison ==, !=, >, <, <=, >= Compare
        assignment =, +=, -=, *=, /=, %=, &=, |=, ^=, <<=, >>= Assigning values to variables
        string concatenation + Joints strings
        type conversion (type), as, is, typeof, sizeof Conversion of data
        other ., new, (), [], ?:, ??

    • Exchange bit n with bit m

      • x = x ^ (~(1 << m)) | (bit at n << m)
