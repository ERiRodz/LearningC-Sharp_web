    • Any base (numeral system) to decimal representation

        1. Count the number of digits or letter used to write the value. For example, 1111 (in binomial) has four ones; FA5 (in hexadecimal) uses three symbols (two letters and one five). From left to right, let's use 'n' as the position of the digit or letter used in the value representation being the one at the right most of them the starting position, n = 0 . For example, in FA5 'n' for 'F' is 2 , for 'A' is n = 1 and for '5' is n = 0.
        2. From left to right, multiply the digit or the letter value by the power, determined by the position of the symbol, n, of the base number used un the numeral system. Then, add all the products. For example, FA5 = 15 × 162 + 10 × 161 + 5 × 160 = 4,005
