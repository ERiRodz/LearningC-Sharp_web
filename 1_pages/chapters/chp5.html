    • Comparing Reference Types

      • When comparing reference types varibales, the addresses they hold are compared. The program checks if both references point to the same location in the memory (same object in the heap). So the comparison operator (==) does not compare the content of the objects but rather checks if they point at the same location in the memory. This is true for all objects types except for strings objects. The comparison operator compares the strings by their values and not by the addresses they point. The size comprasions are not appicable for object type variables.
